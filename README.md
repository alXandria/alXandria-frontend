# Praetor Frontend

Praetor frontend provides new ui for providers in Akash network.

## Run Locally

Clone the project

Please use node 14.15

```bash
  git clone https://github.com/PraetorOne/praetor-frontend.git
```

Go to the project directory

```bash
  cd praetor-frontend
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
