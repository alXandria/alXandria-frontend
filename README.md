# Alxandria Frontend

Alxandria Project

## Run Locally

Clone the project

Please use node 14.15

```bash
  git clone git@github.com:Arc10-Technologies-Inc/alxandria-frontend.git
```

Go to the project directory

```bash
  cd praetor-frontend
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
